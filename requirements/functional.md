User stories for functional reqs:
1. As a user I want to browse meals to see what I can order/make.
2. As a user I want to order ingredients of a meal to be able to make that meal myself.
3. As a user I want to filter meals according to a tag (like vegan) to see specific types of meals.
4. As a user I want to schedule my delivery to set a specific date for it to arrive.
5. As a user I want to find meals that include a specific ingredient to make a meal with ingredients that I already have.
6. As a user I want to search a meal to find the list of ingredients for a specific meal that I want.
7. As u user I want to track my order to see when it will arrive.
