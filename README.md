# SmartCater Prototype

### Functional Requirement Implemented
Browsing meals offered by the catering service, with the ability to search by name, ingredient, or category

### Non-Functional Requirement
The platform is designed to be easy and intuitive to use, with clear prompts and controlled flow

### How to Run the Prototype

1. Python 3 needs to be installed
2. Clone this repository to your local device.
3. Open this repo in your terminal and run the browse_meals.py file.
4. Follow the text instructions given by the file on the terminal
